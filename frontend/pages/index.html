Response Template Documentation

Complete guide to all RAG system response templates

About These Templates

These response templates are used by the RAG (Retrieval Augmented Generation) system to generate intelligent, contextual responses to customer queries. Each template is automatically selected based on semantic similarity matching between the customer query and template keywords.

How it works: When a customer submits a query, the system searches these templates using semantic similarity, retrieves the most relevant ones, and uses them as context for Gemini AI to generate personalized responses.

How Templates Are Used

  1. Customer submits a query via email, form, or chat
  2. NLP classifies the intent and urgency
  3. RAG system searches templates using semantic similarity
  4. Top 3 most relevant templates are retrieved
  5. Gemini AI generates personalized response using templates as context
  6. Response is returned to customer and stored with ticket
