Overview

This response template provides guidance on identity verification issues. It's used when customers experience problems with ID verification, document verification, or verification failures.

Use Cases

  • Document verification issues

  • Verification process problems

Common Scenarios

  • Failed Verification: ID verification keeps failing
  • Document Issues: Documents rejected during verification
  • Blurry Images: Poor quality document photos
  • Expired Documents: Using expired ID documents
  • Name Mismatch: Name on documents doesn't match

Response Template

Template Response

Identity verification requires clear, legible documents. Common reasons for failure: (1) Blurry or cropped images, (2) Expired documents, (3) Name mismatch between documents, (4) Unsupported document types. Please ensure: Valid government-issued ID, Clear photo/scan, All corners visible. If issues persist, our KYC team can review manually.

Response Approach

  1. Requirements: Explain what's needed for verification
  2. Common Issues: List typical reasons for failure
  3. Best Practices: Provide document quality guidelines
  4. Requirements Checklist: List document requirements
  5. Manual Review: Offer manual review if automated fails

Document Requirements

  • Valid ID: Government-issued identification document
  • Clear Image: High-quality, non-blurry photo or scan
  • Complete Document: All corners and edges visible
  • Current: Document must not be expired
  • Name Match: Name must match across all documents
  • Supported Format: Accepted document types only

Common Failure Reasons

  • Blurry or low-quality images
  • Cropped or incomplete documents
  • Expired identification documents
  • Name mismatch between documents
  • Unsupported document types
  • Poor lighting in photos
