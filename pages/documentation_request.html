Documentation Request

documentation_request

Overview

This response template points customers to documentation resources. It's used when customers need help finding guides, tutorials, API documentation, or how-to information.

Keywords

documentation how to guide tutorial api docs

Use Cases

  • API documentation requests
  • Integration guides needed
  • Tutorial or how-to questions
  • Code samples requested
  • Best practices documentation

Common Scenarios

  • API Reference: Need API endpoint documentation
  • Integration Guide: Looking for step-by-step integration instructions
  • Code Samples: Requesting example code or snippets
  • Best Practices: Need guidance on implementation patterns
  • How-To Questions: General how-to questions

Response Template

Template Response

Our comprehensive documentation is available at docs.finlink.com. You can find: (1) API reference, (2) Integration guides, (3) Code samples, (4) Best practices. If you need specific guidance, please let us know what you're trying to accomplish and our Tech Support team can provide targeted assistance.

Response Approach

  1. Direct Link: Provide documentation URL
  2. Content Overview: List what's available in docs
  3. Categories: Mention API reference, guides, samples, best practices
  4. Additional Support: Offer targeted help if needed
  5. Team Assignment: Route to Tech Support for specific questions

Documentation Resources

  • API Reference: Complete API endpoint documentation
  • Integration Guides: Step-by-step integration tutorials
  • Code Samples: Example code in multiple languages
  • Best Practices: Recommended implementation patterns
  • SDKs: Software development kits documentation

When to Use

This template is automatically selected by the RAG system when:

  • Query mentions "documentation", "how to", "guide", "tutorial"
  • Intent is classified as general_support or technical_support
  • Urgency is low to medium
  • Customer is asking informational questions

Team Assignment

Queries matching this template are routed to: Tech Support
